# ✨ AI Paragraph Generator

A web-based paragraph generation application developed using **HTML, CSS, JavaScript, Node.js, and Express.js**. The application allows users to enter a topic and instantly generates a well-structured paragraph through a simple and interactive interface.

---

## 📖 Project Overview

The AI Paragraph Generator is designed to generate informative paragraphs based on a user-provided topic. The application features a modern user interface with an animated background and communicates with a lightweight Express.js backend to produce paragraph content.

---

## ✨ Features

- 📝 Generate paragraphs from a user-entered topic
- 🌐 Interactive web-based interface
- 🎨 Modern animated background design
- ⚡ Fast paragraph generation
- 📱 Simple and responsive layout
- 🔄 Express.js backend for handling requests
- 💻 Easy to use and lightweight application

## 🚀 Installation

### Clone the repository

```bash
git clone https://github.com/hemasree200807/AI-Paragraph-Generator.git
```

### Navigate to the project folder

```bash
cd AI-Paragraph-Generator
```

### Install the dependencies

```bash
npm install
```

### Start the server

```bash
node server.js
```

The server will start on:

```
http://localhost:5000
```

### Open the application

Open **index.html** in your browser.

---

## ▶️ How It Works

1. Open the application.
2. Enter a topic in the input field.
3. Click the **Generate** button.
4. A loading message ("Thinking... 🤖") is displayed.
5. The application generates a paragraph related to the entered topic.
6. The generated paragraph is displayed on the screen.
